# HTML

文档撰写的原因：之前一直认为HTML只是超文本语言，相对比较简单，但是在实际应用中仍然会出现一些问题

参考文档：[HTML 标签列表(字母排序) | 菜鸟教程 (runoob.com)](https://www.runoob.com/tags/html-reference.html)

例如：

```html
<div>
    <ul>
        <li>这是一个li标签</li>
        <span>
            这是一个span标签(错误的用法)
        </span>
        <li>这是第二个li标签</li>
    </ul>
</div>
```

ul中嵌套只能li标签，不能嵌套其他标签或者文字
